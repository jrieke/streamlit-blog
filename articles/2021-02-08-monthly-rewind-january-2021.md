## Streamlit January updates

The latest and greatest that you might have missed in January.

### üîç Current releases: 0.76.0

The newest Streamlit release is [0.76.0](https://discuss.streamlit.io/t/version-0-76-0/9522?ref=streamlit.ghost.io), but make sure to check out the [change log](https://docs.streamlit.io/en/0.76.0/changelog.html?ref=streamlit.ghost.io) if you haven't updated in a while to see what's been going on.

### üïπ Closed betas

We currently have a number of coming soon features that are being tested in closed beta, including:

* Secrets management (for Streamlit sharing)
* Programmable state
* Customizable theming

If you're interested in testing any of these [send us a message on the forum](https://discuss.streamlit.io/?ref=streamlit.ghost.io) and we'll add you to the beta when it becomes available.

### üß© New components

The community continues to actively develop a number of [new components](https://www.streamlit.io/components?ref=streamlit.ghost.io) that you can use to extend your apps. Here are two that made some headway in January:

* **[Streamlit Ag-Grid](https://github.com/PablocFonseca/streamlit-aggrid?ref=streamlit.ghost.io) by Pablo Fonseca** -Streamlit Ag-Grid makes it possible to use interactive dataframes within your Streamlit apps. [[Demo App](https://share.streamlit.io/pablocfonseca/streamlit-aggrid/main/examples/example.py?ref=streamlit.ghost.io)]

![gif-1-2](https://streamlit.ghost.io/content/images/2021/08/gif-1-2.gif#browser)

* **[Streamlit WebRTC](https://github.com/whitphx/streamlit-webrtc?ref=streamlit.ghost.io) by Yuichiro Tsuchiya** -Use Streamlit WebRTC to send and receive video within your Streamlit apps. [[Demo App](https://share.streamlit.io/whitphx/streamlit-webrtc-example/main/app.py?ref=streamlit.ghost.io)]

![gif-2](https://streamlit.ghost.io/content/images/2021/08/gif-2.gif#browser)
