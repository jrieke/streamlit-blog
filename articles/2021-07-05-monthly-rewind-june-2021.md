## **Featured community content**

Some great apps, videos, and articles by the Streamlit community. Check out the [Streamlit forum](https://streamlit.ghost.io/p/efb2b033-a8e4-47ec-bb60-f1313d84052a/discuss.streamlit.io) for even more community content.

* Akshansh created a [CodeGame](https://www.linkedin.com/posts/akshanshkmr_streamlit-python-programming-activity-6805840731933151232-JzzP?ref=streamlit.ghost.io) app where you build high scores by identifying the languages of random code snippets
* [Options-2-trees](https://www.linkedin.com/feed/update/urn:li:activity:6806941263657607168?updateEntityUrn=urn%3Ali%3Afs_feedUpdate%3A%28V2%2Curn%3Ali%3Aactivity%3A6806941263657607168%29&ref=streamlit.ghost.io) is an interactive visualization of the CRR binomial options pricing model by Tony
* Learn how to build a text classifier model and app in Jesse's tutorial [NLP Project - Emotion In Text Classifier App with Streamlit and Python](https://www.youtube.com/watch?v=tLsg01D6k6g&ref=streamlit.ghost.io)
* Billy goes over [How To Create An Instagram Profile Analyzer App Using Python And Streamlit](https://pub.towardsai.net/how-to-create-an-instagram-profile-analyzer-app-using-python-and-streamlit-11107046e85b?ref=streamlit.ghost.io) with instagram-scraper in his blog post
* If you want to see how your Medium posts are performing, Rahul wrote about [How to get a JSON file of your Medium Stats and create a Dashboard using Streamlit](https://www.realpythonproject.com/how-to-get-a-json-file-of-your-medium-stats-and-create-a-dashboard-using-streamlit/?ref=streamlit.ghost.io)
* Label intent, along with entity recognition and entity frequency for your GSC queries with Greg's [GSC Query Analysis Tool](https://twitter.com/GregBernhardt4/status/1403344512848646146?s=20&ref=streamlit.ghost.io)
* Find your [Hacker News Doppelg√§nger](https://twitter.com/Pinecone_io/status/1405991431764840448?s=20&ref=streamlit.ghost.io) with this app by Pinecone, which compares the semantic meaning of your comment history with those of all other users, and finds the top ten users whose comment histories are most similar to yours
* [Solar-MACH](https://twitter.com/JanGieseler/status/1405559208079069184?s=20&ref=streamlit.ghost.io) is an awesome app by Jan and crew that plots multi-spacecraft longitudinal configuration
