## ****Streamlit** February u**pdates****

Below are the latest updates and releases from February.

### üîç Current release: 1.20.0

The latest release is 1.20.0. Recent updates include granular control over app embedding behavior, a cleaner hamburger menu, and the de-experimentalization of faster reruns. Check out the [changelog](https://docs.streamlit.io/library/changelog?ref=streamlit.ghost.io) to learn more about the latest features and fixes.

### ****üîÆ** Upcoming**

New features to get excited about:

* Step parameter for `st.time_input`
* Tooltips on text elements
* Improved `st.help`

Visit our [roadmap app](https://roadmap.streamlit.io/?ref=streamlit.ghost.io) to see what else we're working on. ü•≥

### **‚ö°Ô∏è New caching commands**

* Introducing `st.cache_data` and `st.cache_resource`‚Äîtwo new simpler and faster commands to replace `st.cache`! Learn more about them [here](https://streamlit.ghost.io/introducing-two-new-caching-commands-to-replace-st-cache/).

* Get scientific searches and insights all in one place with Avra's [PubLit](https://twitter.com/Avra_b/status/1620606960709300224?s=20&t=HAVoAqp4wI-yK3NMBwMEoQ&ref=streamlit.ghost.io)‚Äînow using GPT3.

* Present your portfolio and projects as visually pleasing cards with Tom's [Streamlit Cardfolio: A Portfolio Presentation App!](https://discuss.streamlit.io/t/streamlit-cardfolio-a-portfolio-presentation-app/37540?ref=streamlit.ghost.io)
