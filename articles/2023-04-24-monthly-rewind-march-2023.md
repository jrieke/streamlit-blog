This app allows you to extract vocal and instruments from any audio. It uses a pre-trained model called Hybrid Spectrogram and Waveform Source Separation. ## ****Streamlit** March u**pdates****

Below are the latest updates and releases from March.

### üîç Current release: 1.21.0

The latest release is 1.21.0. Recent updates include improved `st.help`, support of global `secrets.toml` file, and a `step` parameter for `st.time_input`. Check out the [changelog](https://docs.streamlit.io/library/changelog?ref=streamlit.ghost.io) to learn more about the latest features and fixes.

### ****üîÆ** Upcoming**

New features to get excited about:

* Easily connect to data sources with `st.experimental_connection`
* Customize the visibility items in the toolbar, options menu, and the settings dialog
* Ability to embed Streamlit apps

Visit our [roadmap app](https://roadmap.streamlit.io/?ref=streamlit.ghost.io) to see what else we're working on. ü•≥
